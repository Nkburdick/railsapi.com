Features

  • Browse through classes and modules

  • Smart match

  • Recent searches (only in Safari)

  • Find files

  • Refine with namespace

  • Keyboard navigation

    Up Arrow Key Move to previous result
    Down Arrow Key Move to next result
    Enter Open selected result
    Right Arrow Key Open selected folder
    Left Arrow Key Close selected folder or move to parrent folder

    Hold arrow key to move selection faster
